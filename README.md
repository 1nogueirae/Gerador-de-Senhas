# 🔐 Gerador de Senhas

Um gerador de senhas seguro e personalizável desenvolvido com HTML, CSS e JavaScript puro.

## ✨ Funcionalidades

- **Geração de senhas seguras** com até 64 caracteres
- **Personalização completa** com opções para:
  - Letras maiúsculas
  - Números
  - Símbolos especiais
- **Interface intuitiva** com slider para ajustar o tamanho
- **Cópia rápida** da senha gerada
- **Design responsivo** e moderno

## 🚀 Como usar

1. Acesse o gerador no seu navegador
2. Ajuste o tamanho da senha usando o slider (4-64 caracteres)
3. Marque/desmarque as opções desejadas:
   - ✅ Maiúsculas (A-Z)
   - ✅ Números (0-9) 
   - ✅ Símbolos (!@#$%...)
4. Clique no botão de refresh para gerar uma nova senha
5. Use os botões de copiar para usar a senha

## 🛠️ Tecnologias

- **HTML5** - Estrutura semântica
- **CSS3** - Estilos modernos com Flexbox
- **JavaScript ES6** - Lógica de geração e interatividade

## 🔧 Instalação

1. Clone ou baixe os arquivos
2. Abra o `index.html` no navegador
3. Pronto! Não precisa de servidor ou dependências

## 💡 Características técnicas

- Algoritmo de geração verdadeiramente aleatório
- Garantia de inclusão de pelo menos um caractere de cada categoria selecionada

- Compatível com navegadores modernos
